XEB data for SQ and CZ gates.

----------------
| Independents |
----------------
ms          : XEB gate numbers

--------------
| Dependents |
--------------                     
q\*_xeb_fid : XEB fidelities of SQ gates
c\*_xeb_fid : XEB fidelities of CZ gate

-------------
| Constants | 
-------------    
q\*_error_pauli : single-qubit gate Pauli errors
c\*_error_pauli : CZ gate Pauli errors

---------- 
| suffix |
----------
_fit      : fitted values
